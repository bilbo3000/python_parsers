Release Notes

0.8.2

Improvements in this release:

  • Now works with Firefox 2.0.0.* and Thunderbird 2.0.0.*

0.8.2

Improvements in this release:

  • Now works with Firefox 1.5.0.* and Thunderbird 1.5.0.*

0.8

Improvements in this release:

  • Now works with Firefox 1.5 and Thunderbird 1.5.
  • Thanks to Matthias Cramer, Robert Sack, Doug Hays, Jay, Damon Overboe, Tacvek, Adam Guthrie, Kevin Fuller and Casey Parker for help in refining the preferences list.

0.7.1

Improvements in this release:

  • IMPORTANT: Changed GUID to valid format. This means that people using Firefox or Thunderbird must uninstall Preferential before installing this update.
  • Fixed tree handling code to work with more recent versions of Mozilla. Possible fix to bug

0.7

Improvements in this release:

  • Now compatible with Firefox/Thunderbird Extension Manager.
  • More preferences documented.

0.6.1a

Improvements in this release:

  • Addition of tool to generate JavaScript code to append to user.js.
  • More preferences documented, particularly in the ldap_1.*, ldap_2.* and mail.* areas.

0.6

Improvements in this release:

  • Over 650 preferences documented! Many thanks to Mike C and all other contributors.
  • Implemented 'proper' progress meter.
  • Implemented wildcarding of preferences to allow proper documentation of preferences even when 'generated' by the user.
